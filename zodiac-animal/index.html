📖 사용 방법
  1. 양력 생년월일을 선택하세요
  2. 음력 설 날짜를 자동으로 비교하여 정확한 띠를 계산합니다
  3. 1~2월생의 경우 설 이전이면 전년도 띠로 자동 보정됩니다
  4. '결과 복사' 버튼으로 텍스트 결과를 복사할 수 있습니다

💡 십이지신은 12년을 주기로 순환합니다. 천간(10개)과 지지(12개)가 조합되어 60갑자를 이루며, 60년마다 같은 간지가 돌아옵니다.
