UTM 파라미터란?

URL 뒤에 붙는 추적 코드로, Google Analytics 등에서 방문자가 어떤 경로로 들어왔는지 분석할 때 사용합니다.


각 파라미터 설명
  • utm_source — 트래픽 출처 (어디서 왔는지). 예: google, facebook, naver
  • utm_medium — 유입 매체 (어떤 방식으로). 예: cpc(클릭광고), email, social, banner
  • utm_campaign — 캠페인 이름 (왜 왔는지). 예: spring_sale, new_product_launch
  • utm_term — 검색 키워드 (유료 검색광고에서 사용)
  • utm_content — 같은 캠페인 내 콘텐츠 구분 (A/B 테스트 등)

작성 팁
  • 소문자로 통일하세요 (Google과 google은 다르게 집계됩니다)
  • 공백 대신 _(밑줄) 또는 -(하이픈)을 사용하세요
  • 팀 내에서 일관된 네이밍 규칙을 정해두면 분석이 편합니다
  • 프리셋을 활용하면 자주 쓰는 source/medium을 빠르게 입력할 수 있습니다
