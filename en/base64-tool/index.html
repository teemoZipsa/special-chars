📖 How to Use

Text Mode:

• Encode: Converts normal text into a Base64 string.

• Decode: Restores a Base64 string back to original text.

• Fully supports Unicode characters including emojis (UTF-8).


Image Mode:

• Converts an image file into a Base64 Data URL.

• Can be used as inline images in HTML/CSS.


⚠️ 100% Client-side Processing — no data is sent to the server.
