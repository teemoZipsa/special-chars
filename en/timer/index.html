💡 Timer / Stopwatch Guide

How to use the Timer
Set the hours, minutes, and seconds using the input fields, or instantly set a time using the quick buttons (e.g., "5m", "1h"). Click the Start button (▶) to begin the countdown. An alarm will sound automatically when the time is up.
How to use the Stopwatch
Switch to the Stopwatch tab and click Start (▶). While it's running, you can click the Lap button (🏁) to record lap times and see the difference between each lap. Press the Stop button (■) when you're finished.
Handy Keyboard Shortcuts
  • Spacebar: Start / Stop
  • R: Reset the timer or stopwatch
  • L: Record a lap (Stopwatch mode only)
