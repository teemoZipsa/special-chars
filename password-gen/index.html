🤔 안전한 비밀번호 Q&A
안전한 비밀번호란 무엇인가요?
컴퓨터 시스템과 온라인 계정을 해킹으로부터 보호하기 위해 추측하기 어렵고(무작위성) 기계적인 암호 대입 공격(Brute Force Attack)을 견뎌낼 수 있게 만든 비밀번호입니다. 대문자, 소문자, 숫자, 특수문자를 혼합하고 12자리 이상의 길이로 설정하는 것이 국제 표준 보안 권장 사항입니다.
내 비밀번호는 서버에 저장되나요?
아니요. 절대 저장되지 않습니다. 저희 도구는 100% 사용자의 브라우저 로컬 환경(클라이언트 사이드)에서 crypto.getRandomValues() 함수를 사용하여 비밀번호를 생성합니다. 외부 네트워크로 전송되는 데이터가 없으므로 완전히 안전하게 사용하실 수 있습니다. 브라우저를 닫으면 최근 생성 기록도 모두 삭제됩니다.
최근 생성 기록은 어떻게 활용하나요?
여러 사이트에 가입하거나 복수의 비밀번호가 필요할 때, 미리 여러 개를 생성해 두고 클릭 한 번으로 복사하여 빠르게 사용할 수 있도록 돕는 임시 보관함입니다.
