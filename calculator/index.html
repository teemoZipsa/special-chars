🧮 계산기

0
📖 사용 방법
  1. 화면의 버튼을 클릭하거나 키보드로 직접 숫자와 연산자를 입력하세요.
  2. C 버튼으로 전체 초기화, ← 버튼으로 마지막 입력을 삭제합니다.
  3. % 버튼으로 퍼센트 계산, = 또는 Enter로 결과를 확인합니다.
💡 자주 묻는 질문
Q. 키보드로 계산기를 조작할 수 있나요?

네, 키보드 입력을 완전히 지원합니다. 숫자(0~9), 연산자(+, -, *, /), 소수점(.), Enter(=), Escape(초기화), Backspace(마지막 입력 삭제), %(퍼센트)를 키보드로 직접 입력할 수 있습니다. 마우스 클릭 없이 빠르게 연속 계산이 가능합니다.


Q. 퍼센트(%) 계산은 어떻게 하나요?

% 버튼은 현재 입력된 숫자를 100으로 나눕니다. 예를 들어 "200 × 15 %" = 30 (200의 15%), "500 + 10 %" = 550 (500에 10% 추가)과 같이 활용할 수 있습니다. 할인율, 세금 계산 등에 유용합니다.


Q. 계산 결과를 복사하거나 기록할 수 있나요?

현재 버전에서는 별도의 계산 기록 저장 기능은 제공하지 않습니다. 계산 결과는 화면에 표시되며, 필요 시 직접 메모하시거나 스크린샷을 활용해 주세요. 연속 계산은 이전 결과에 이어서 연산자를 입력하면 바로 이어서 계산할 수 있습니다.
