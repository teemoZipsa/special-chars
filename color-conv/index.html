📖 사용 방법
  1. 컬러 피커를 클릭하여 원하는 색상을 시각적으로 선택하세요
  2. 또는 HEX / RGB / HSL 입력란에 직접 코드를 입력하세요
  3. 어느 한 곳을 변경하면 나머지 형식이 자동으로 변환됩니다
  4. 각 입력란의 '복사' 버튼으로 원하는 형식을 복사하세요
  5. 하단의 관련 색상(보색, 유사색)을 클릭하면 해당 색상으로 전환됩니다

💡 HEX: 웹 개발에서 가장 많이 사용. RGB: CSS 함수 형태. HSL: 색상(Hue)·채도(Saturation)·명도(Lightness) 기반으로 직관적 조절에 유리합니다.
