🤔 색상 코드 변환기 관련 Q&A
HEX 코드와 RGB 코드는 어떤 차이가 있나요?
HEX 코드는 16진수로 색상을 표현하는 방식으로, 웹(HTML/CSS) 디자인에서 가장 보편적으로 사용됩니다. (예: #533483)
RGB 코드는 빛의 3원색인 Red, Green, Blue를 각각 0~255의 숫자로 나타내어 화면에 출력하는 방식입니다. 사진 보정, 모니터 출력 등에 주로 사용됩니다.
HSL은 언제 주로 사용하나요?
HSL (Hue, Saturation, Lightness)는 '색상, 채도, 명도'로 구성된 단위입니다.
직관적으로 명도나 채도를 증감시키기 편해, 웹 접근성(대비)을 조절하거나 다크 모드용 컬러 팔레트를 만들 때 효율적으로 활용됩니다.
관련 색상의 '보색'이란 무엇인가요?
보색(Complementary Color)은 색상환에서 서로 마주보고 있는 반대되는 색을 말합니다.
디자인에서 강렬한 대비를 주고 싶을 때나, 핵심 버튼(CTA)의 컬러를 선정할 때 보색이나 유사색을 참고하면 조화로운 배색이 가능합니다.
